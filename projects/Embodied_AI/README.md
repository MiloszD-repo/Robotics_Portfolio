This project focuses on inteligent embodied design of machines that helps with complepleting their functions and allows for simplification of the control (software) part. The task was to develop a mobile robot capable of search and rescue.
The platform used was Lego Mindstorms. The core problems of the project were line following, incline driving, object detection and object grasping. Th line following task was overcome utilizing a regulation system (color mapping). 
Grasping system was completly mechanical utilizing stored elasitc energy.
